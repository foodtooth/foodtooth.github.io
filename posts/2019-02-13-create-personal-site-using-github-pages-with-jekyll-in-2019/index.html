Create personal site using Github Pages with Jekyll in 2019

September 20, 2019 One minute read
static site
jekyll • github pages

This tutorial uses User and Organization Pages for general usage personal site

Create github pages repo

Follow instructions from GitHub Pages, to create a user site repo (Remember to initialize it with a README.md file). Here we use my own repo foodtooth.github.io for simplicity.

Generate default jekyll site

  1. Install Jekyll. Refer to jekyll installation doc
  2. git clone https://github.com/foodtooth/foodtooth.github.io
jekyll new --force foodtooth.github.io
cd foodtooth.github.io

  3. Follow instructions in Gemfile to enable gem "github-pages"

  4. bundle update

  5. Copy the content of this gitignore to .gitignore file

  6. bundle exec jekyll serve to start jekyll on local

Now, you are running a default jekyll site on local. Feel free to do the editing and push the repo to github when you are ready.

Further instructions

If you hesitate about choosing the right theme to use, pick minimal-mistakes. Beautiful and simple looking, detailed and nice documentation. After walking through the documentation and doing some editings, your personal blog is good to go.
